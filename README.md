##关于
    用于Android开发的一些常用工具类
    Licence: Apache-2.0

##功能
1. 一些常用的加密算法：AES, DES, RSA, MD5等
2. 使用ZBar进行二维码、条形码等扫描
3. 使用ZXing进行二维码、条形码扫描

##怎样使用
1. 使用加密算法库:

		compile 'com.hjy.library:encryption:1.0.0'	
2. 使用ZBar进行条形码扫描
		
		compile 'com.hjy.library:zbar:1.0.0'

3. 使用ZXing进行条形码扫描
		
		compile 'com.hjy.library:zxing:1.0.0'
